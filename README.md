# 📡 EcoPulse — Macro-Driven Market Analytics Dashboard

## 🚀 Overview

EcoPulse is an **investor-grade analytics dashboard** that predicts market stress in India and recommends **optimal sector allocations** using macroeconomic indicators.

It combines **data science, financial modeling, and portfolio optimization** into a single interactive Streamlit application.

---

## 🎯 Problem Statement

Financial markets are highly sensitive to macroeconomic changes such as:

* Interest rates (Repo Rate)
* Inflation (CPI)
* Currency movements (USD/INR)
* Market volatility (India VIX)

Investors often struggle to:

* Detect early signs of market stress
* Adjust portfolios dynamically
* Understand macro impact on sector performance

👉 EcoPulse solves this by providing a **data-driven early warning system + portfolio optimizer**.

---

## 💡 Key Features

### 🔴 Market Stress Prediction

* Uses ML model (XGBoost/Sklearn)
* Predicts probability of market stress
* Identifies stress vs normal regimes

### 📊 Sector-Level Insights

* BUY / HOLD / REDUCE signals
* Based on macro indicators
* Covers:

  * Bank Nifty
  * Nifty IT
  * Nifty Pharma
  * Nifty Auto
  * Nifty FMCG

### 💼 Portfolio Optimisation

* Max Sharpe Ratio
* Minimum Volatility
* Risk Parity
* Regime-Adaptive Strategy
* Equal Weight

### 🌐 Macro Scenario Engine

* Simulate real-world events:

  * War / Crisis
  * Rate hikes
  * Currency crash
  * Recession
* Get:

  * Stress probability
  * Recommended allocation

### 📈 Risk Analytics

* VaR / CVaR
* Drawdowns
* Correlation matrix
* Stress vs Normal return distribution

### 💵 Investment Calculator

* Lump sum & SIP modes
* Forecast future portfolio value
